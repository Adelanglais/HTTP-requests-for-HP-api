# HTTP-requests-for-HP-api

Code permettant d'obtenir des observations à partir d'une API.
API utilisée : ratp
Format : XML

## Installation
```bash
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```
# Usage
```python
pythonn3 getSpells.py # returns dictionnaire des sortilèges
```
# Quit Virtualenv
```bash
deactivate
```
